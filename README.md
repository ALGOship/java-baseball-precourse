# 기능 목록

- 컴퓨터의 수 생성 (게임시작)
  - 각자리 수는 1~9까지의 수
  - 서로 다른 수 3자리수
- 플레이어의 수 <u>입력 받기</u>
  - 각자리 수는 1~9까지의 수
  - 서로 다른 수 3자리수
- 컴퓨터&플레이어의 수 비교 
  - 스트라이크
  - 볼
  - 낫싱
- 정답 판단 (게임종료)
  - 3스트라이트
- 게임 진행여부 <u>입력 받기</u>
  - 1 : 새로운게임시작(=컴퓨터의 수 생성)
  - 2 : 종료
> 입력 받기의 경우, [ERROR]로 시작하는 에러 메시지를 출력하고 게임을 계속 진행
